<div className="mt-3 max-w-[560px] text-sm leading-relaxed text-muted-foreground">

## Generate the schema

<div className="mt-3 max-w-[560px] text-sm leading-relaxed text-muted-foreground">
   Run the CLI to generate the schema file for your adapter and provider. Pass the output path where
   you want the file written.
</div>

<div className="mt-5 rounded-xl border border-border bg-card overflow-hidden">
   <CodeBlock lang="bash" code={`npx @authio/cli generate --output ./src/db/auth-schema.ts`} />
</div>

<div className="mt-4 max-w-[560px] text-sm leading-relaxed text-muted-foreground">
   Then push the schema to your database.
</div>

<div className="mt-5 rounded-xl border border-border bg-card overflow-hidden">
   <CodeBlock lang="bash" code={`npx drizzle-kit push`} />
</div>
